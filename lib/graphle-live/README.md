# Graph Live

`@dpeek/graphle-live` owns generic live-scope request contracts, client refresh
control, and server-side invalidation routing.

The root package exports the live-scope request/response contracts, request
helper, and refresh controller. The `server` subpath exports the generic
live-scope router.

## Entrypoints

- `@dpeek/graphle-live`
- `@dpeek/graphle-live/server`
